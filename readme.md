# dWebX Node
This library has been deprecated. If you're looking to build dWeb-based apps using Node, here are some options:
- [dWebSwarm](https://github.com/distributedweb/dwebswarm)
- [dWebSwarm DHT](https://github.com/distributedweb/dht)
- [dWebSwarm Discovery](https://github.com/distributedweb/discovery)
- [dWebSwarm Network](https://github.com/distributedweb/network)
- [dDrive](https://github.com/distributedweb/ddrive)
- [dDatabase](https://github.com/distributedweb/ddatabase)
- [dDrive Daemon Client](https://github.com/distributedweb/ddrive-daemon-client)
